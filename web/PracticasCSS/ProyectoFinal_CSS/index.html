Decidí retomar parte de mi proyecto final de html y darle continuidad con lo que vaya aprendiendo de css.

¿Quién soy?

Soy estudiante de sistemas, actualmente aprendiendo HTML y CSS para esta materia. No es la primera vez que trabajo en desarrollo web, pero estos cursos me está ayudando a reforzar cosas que ignoraba.

Domina para transformar

— Diego de Granda, Platzi

Lo que más ha llamado mi atención por modulo:

Modulo 1

➕ Positions
Aprender que normalmente se usa fixed para los chatbot de ayuda me resultó enriquecedor.
🧐 Uso de clases
Mantienen todo más ordenado además de ser una gran herramienta para optimizar el diseño.
🥡 Box model
Lo tomé como el interlineado y herramientas que ayudan a seccionar contenido.
👁️‍🗨️ Uso de unidades modernas
Importante para que mis trabajos sean incluyentes y accesibles.

Modulo 2

🖋️ Google Fonts
Poder añadir fuentes externas y seguras le puede llegar a dar otro aire a nuestra pagina web.
✅ Control de texto
Es importante manejar la indentación; una pagina con todo el texto a la izquierda no es atractivo.

Modulo 3

⛓️ Variables CSS
Centralizar colores para optimizar cambios y mantener un código limpio.
☁️ Sombras y Bordes
Dar profundidad con box-shadow y suavizar elementos con border-radius.

Modulo 4

🎨 Colores y Backgrounds
Manejo de transparencias RGBA y propiedades de fondo para evitar deformaciones.

Modulo 5

📱 Responsive Design
Uso de Media Queries para que la interfaz reaccione a diferentes dispositivos.

Modulo 6

✨ Efectos y Animaciones
Dar vida a los elementos con transiciones y transformaciones para mejorar la experiencia de usuario.

Implementación Técnica: Módulos 1 al 3

Sticky & BEM

Navegación fija y clases estructuradas:

.header { position: sticky; }

Tipografía

Fuentes externas y legibilidad:

font-family: 'Archivo Black';

Variables CSS

Centralización de estilos:

--accent: #b9104b;

Implementación Técnica: Módulos 4 al 6

Gradients & RGBA

Degradados y transparencias:

linear-gradient(135deg, #861e8a, #f63bb2);

Media Queries

Adaptación a móviles:

@media (max-width: 768px) { ... }

Animaciones

Efectos visuales y transiciones:

transition: transform 0.3s ease;
