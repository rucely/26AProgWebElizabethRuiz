Implementación Técnica: Módulo 1

Pseudo-elementos ::before

Contenido dinámico para las comillas:

.quote p::before { content: '"'; color: #b9104b; }

Positioning sticky

Header fijo durante el scroll:

.header { position: sticky; top: 0; z-index: 1000; }

Metodología BEM

Clases claras (Bloque__Elemento):

.learning-list__term { color: var(--accent); }

Implementación Técnica: Módulo 2

Google Fonts

Importación y uso de fuentes externas:

font-family: "Archivo Black", sans-serif;

Propiedades de Fuente

Control de tamaño y peso visual:

.hero__title { font-size: 4rem; font-weight: bold; }

Control de Texto

Alineación y formato del contenido:

.hero { text-align: center; }
